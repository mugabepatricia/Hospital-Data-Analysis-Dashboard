# Hospital-Data-Analysis (MS Excel interactive dashboard)
## Project Overview
This project is an end-to-end hospital data analytics workflow built using Microsoft Excel, covering data cleaning, transformation, analysis, and interactive dashboard creation.
The goal was to explore hospital patient data and generate meaningful insights around patient demographics, diagnoses, treatment patterns, doctor performance and operational efficiency using Excel-based analytics tools.
## Objectives
- Clean and prepare raw hospital datasets for analysis
- Perform exploratory data analysis (EDA)
- Identify key healthcare trends and patterns
- Build interactive dashboards for decision-making
- Present insights in a clear and visual format

- Basic statistical analysis functions
## Data Cleaning & Preparation
The dataset was cleaned using Excel and Power Query by:
- Removing duplicates and blank rows
- Standardizing inconsistent formats (dates, categories, etc.)
- Handling missing values appropriately
- Structuring data into analysis-ready tables
- Ensuring data types were correctly assigned
## Key Analysis Areas
The analysis focused on:
- Patient demographics (gender distribution)
- Common diagnoses and conditions
- Revenue per department
- Revenue per diagnosis
- Length of stay analysis
- Insurance coverage distribution
- Doctors’ performance
## Dashboard Features
The final dashboard includes:
- Interactive filters (slicers) for dynamic analysis
- KPI cards summarizing key hospital metrics
- Trend charts for admissions over time
-	Condition distribution visuals
- Patient demographic breakdowns
- Clean and user-friendly layout for stakeholders
## Final Insights
- The hospital experienced clear variation in patient admissions across different genders and age groups, with certain age brackets consistently showing higher healthcare demand, indicating targeted care needs for those populations.
- A small number of medical conditions accounted for a large proportion of total admissions, suggesting concentrated resource usage around specific diagnoses.
- Patient length of stay varied significantly depending on diagnosis type, highlighting differences in treatment complexity and recovery time.
- Insurance coverage was uneven across patient categories, pointing to potential disparities in access to healthcare services or differences in demographic coverage patterns.
- Overall, the dashboard revealed that patient demographics, diagnosis type, and insurance status are key drivers of hospital resource utilization and patient flow and other insights
